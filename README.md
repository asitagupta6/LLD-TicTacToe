Low-Level Design (LLD) for a Tic-Tac-Toe game

1. Requirements
   
Functional
2 players play turn by turn
Board size: default 3×3 (extendable to NxN)
Detect winner / draw
Prevent invalid moves

Non-Functional
Clean OOP design
Extensible (N players, different board sizes)
Testable

Game
Board
Player
